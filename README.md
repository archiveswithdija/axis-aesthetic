# Curvo 📈

Curvo is a lightweight, interactive graphing interface built using vanilla HTML, CSS, and JavaScript.  
It focuses on dynamic graph management, modular UI design, and a neon-themed visual system.

---

## ✨ Features

- Add and remove graphs dynamically
- Modular graph engine architecture
- Theme system (Neon mode implemented)
- Clean sidebar-based UI
- Organized state management system
- Built with performance and simplicity in mind

---

## 🧠 Tech Stack

- HTML5
- CSS3 (CSS Variables for theming)
- JavaScript (ES Modules)

## 🎨 Design Philosophy

Curvo was built with a focus on:

- Clean modular architecture
- Separation of logic, UI, and state
- Expandability for future graph features
- Aesthetic-first UI (Neon theme system)
